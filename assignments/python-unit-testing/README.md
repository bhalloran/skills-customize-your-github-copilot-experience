# 📘 Assignment: Unit Testing in Python

## 🎯 Objective

Learn how to write and run automated tests for Python code using `pytest`, including test functions, assertions, and test organization.

## 📝 Tasks

### 🛠️ Write Tests for Existing Code

#### Description

Create unit tests for a Python module that implements simple utility functions. Use `pytest` to verify behavior, handle expected errors, and check edge cases.

#### Requirements
Completed project should:

- Include a `tests/` directory with at least three test files
- Cover functions using assertions for correct return values and expected exceptions
- Test normal cases, boundary conditions, and invalid input
- Use `pytest` conventions so tests can be run with `pytest`
- Provide a short `README.md` section explaining how to install dependencies and run tests

### 🛠️ Add Starter Utility Code

#### Description

Provide a simple Python module with functions students can test, such as string formatting, list filtering, and number validation.

#### Requirements
Completed project should:

- Include a `starter-code.py` module with at least three functions
- Use clear function names and docstrings
- Keep implementation simple but non-trivial enough to require tests
- Ensure the module is importable from test files
